# CF Legends Tactical Mappings

An interactive, web-based tactical mapping tool designed specifically for Crossfire Legends. This tool allows IGLs and teams to visually plan strategies, player positioning, and utility deployments.

🌐 **[Access the Live Tool Here](https://Liveon3.github.io/tactical-map/)**

## 🎯 Current Features

* **Mobile-Accurate Maps:** Uses CF Legends specific map layouts (e.g., Black Widow).
* **Offense/Defense Toggle:** Instantly swap token colors depending on whether you are planning a Black List (BL) execute or a Global Risk (GR) defense.
* **Draggable Player Roles:** Spawn specific roles (IGL, Sniper, Lurk, Support, Entry) and position them on the map.
* **Utility Spawner:** Track your team's economy and executes with draggable Smoke, Flash, HE, Molotov, and C4 tokens.
* **Live Drawing Canvas:** Use the freehand green marker to draw entry paths, sightlines, and rotation routes.
* **Text Annotations:** Drop custom text callouts directly onto the map.
* **Undo (Ctrl+Z):** Instantly erase accidental drawings without clearing the whole board.

## 🛠️ How to Use

1. **Select a Map:** Use the dropdown on the top left.
2. **Set the Mode:** Click the Offense/Defense button to set your team's starting side.
3. **Spawn Elements:** Click any button in the sidebar to spawn a player or utility token in the center of the map.
4. **Position:** Click and drag the tokens to their intended positions.
5. **Draw:** Click and drag anywhere on the map to draw paths. Avoid clicking directly on tokens while drawing.
6. **Annotate:** Click "+ Text Note", type your callout, and drag it into position.
